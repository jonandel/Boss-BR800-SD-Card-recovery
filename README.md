# Boss-BR800-SD-Card-recovery
Finds the Headers on a SD Card from a Boss BR800, to identify 'lost' recordings

I was fed up with recording Studio practices and Gigs, and the power being pulled from the device too early before the card had Written the audio as a Sketch,
or on occaision, the BR800 went into 'standby' mode after a long time, and again, simply didnt write the final file to the card.

The code is designed for Windows, and needs to be run as Administrator.
